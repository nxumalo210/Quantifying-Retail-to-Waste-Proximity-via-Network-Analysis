# GIS_Macy-s_Landfill_Project
This project was meant to develop a map of the macy's locations and their proximity to Landfills around New York State. 

Link to project on ArcGIS Online:[https://arcg.is/1DKmmq3]

**Results:**
After conducting a network analysis, only half of the plotted locations are in a 15 mile radius of municipal solid waste landfills around New York State.
